| 场景                     | 推荐                                                                 |
| ------------------------ | -------------------------------------------------------------------- |
| **资金充足,追求增长**    | 主用 **0-4**                                                         |
| **保守或想平滑收益曲线** | 主用 **2-4**                                                         |
| **若要动态切换**         | 在趋势强(高波动)期切 **0-4**,盘整期切 **2-4**,保留 0-2 仅作过渡测试. |
