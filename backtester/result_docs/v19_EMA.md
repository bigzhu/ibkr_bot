## ema 1m 244 附近是较好的买入点, 下跌远离就不适合买入了

与 EMA 的乖离率 0.24% 均可买入, 超过就禁止买入

```python
    # EMA
    ema_price = ema_with_binance_api(
        symbol,
        "1m",
        244,
    )
    deviation = (ema_price - price) / ema_price
    deviation_percentage = deviation * Decimal("100")
    if deviation > Decimal("0.0024"):
        raise ValueError(f"乖离率过大: {format_percentage(deviation_percentage)}")
    if deviation > Decimal("0"):
        entry_amount = entry_amount
    else:
        entry_amount = entry_amount * Decimal("1.4")
```
